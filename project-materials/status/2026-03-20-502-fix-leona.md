# 502 排查与修复记录

**时间：** 2026-03-20 06:49 UTC  
**负责人：** Leona (backend-leona)

## 502 根因

`office-console` 后端服务（pm2 进程）未运行，端口 3014 无监听。
根本原因：`ecosystem.config.js` 使用 CommonJS `module.exports` 语法，但 `package.json` 设置了 `"type": "module"`，导致 pm2 加载 ecosystem 配置时报 ES Module 错误，进程未能启动。

## 修复动作

1. 将 `ecosystem.config.js` 复制为 `ecosystem.config.cjs`（CommonJS 扩展名，绕过 ES Module 限制）
2. 执行 `pm2 start ecosystem.config.cjs`
3. 服务成功启动，pid 1765921，端口 3014 正常监听
