# 首页环境状态标签约定

- 模块/接口名称：首页环境状态标签
- 输入：
  - 首选输入：`GET /api/console/summary` 中的首页状态摘要数据
  - 状态枚举：`pass` / `warn` / `fail` / `unknown`
  - 最小字段来源：模型状态、飞书状态、浏览器状态、Workspace 状态
- 输出：一个首页顶部可见的单标签状态区块，包含状态文案和最小动作提示
- 约束：
  - 首版只允许单标签展示
  - 不含趋势图、悬浮说明、历史记录和多环境切换
  - 文案必须可被办公用户直接理解
  - 标签状态必须与颜色一一对应
- 依赖关系：
  - 依赖产品定义首页顶部状态摘要区语义
  - 依赖环境检测状态标准 `pass / warn / fail / unknown`
  - 依赖前端按状态渲染颜色、文案和最小动作入口
- 备注：
  - 文案映射建议：
    - `pass` -> 环境正常
    - `warn` -> 环境需关注
    - `fail` -> 环境异常
    - `unknown` -> 尚未检测
