1. **Inspector 侧边栏默认 closed**：HTML 加 `class="inspector-panel closed"`，width:0 隐藏，点击才展开
2. **Memory 文件名折行**：`.memory-file-name` 改为 `white-space:nowrap + overflow:hidden + text-overflow:ellipsis + flex:1 + min-width:0`
3. **健康状态时间改北京时间**：全局 `formatChinaDateTime()`，`toLocaleString('zh-CN',{timeZone:'Asia/Shanghai',hour12:false})`；`checkedAt`/`source` 等原始字段映射为中文
4. **全面文本排查**：概览折行/调试头隐藏/标题栏原始字段/`cached`→已缓存/`lastCheck`→检查时间
5. **整体字体上调**：正文 14→15px，标签 12→13px，KPI 36→38px
6. **记忆文件标题分行**（主标题独行+副标题独行）、更新时间格式缩短（`MM-DD HH:mm`）、overflow:hidden 清除
7. **Connection Health "去配置"按钮**：历经多次修复，最终改用 `onclick="window.location.hash='settings'"` 触发 hashchange 路由（最可靠方式）
8. **Settings 页飞书告警通知卡片**：落地配置 `FEISHU_WEBHOOK_URL`，告警阈值：上下文压力≥80%、Agent 空闲>2h
9. **Azir Agent 状态误判**：`getAgentLastActiveMs()` 现在排除 `probe-*.jsonl` 文件（系统探针），避免 false working 状态
10. **浏览器缓存问题**：静态资源从 `max-age=0+ETag` 改为 `Cache-Control: no-store`，强制浏览器不缓存 JS/HTML

### 关键发现
- `app.js` 是 `type="module"` 加载，函数 inline onclick 必须通过 `window.xxx` 挂载才能调用
- `probe-*.jsonl` 文件由 OpenClaw 系统健康检查写入，不代表 Agent 真实工作
- Azir(main) 的 sessions.json 每次用户发消息都会更新（消息路由），这是正常行为
