# 主工作区身份定义

## 作用

这个文件用于定义 `main` 主工作区对应 agent 的展示身份。

它主要用于：

- 名称展示
- 表情展示
- 主题展示
- 头像来源说明

## 说明

- 这里记录的是展示身份，不是运行时路由规则
- 如果后续使用 `openclaw agents set-identity --from-identity`，可以从这里读取
